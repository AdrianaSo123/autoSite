# Sprint 55 - Suggestion Limits and UX Refinement

## Objective
Keep suggested actions useful and uncluttered.

## Tasks
- Enforce maximum of three suggestions.
- Hide stale suggestions once user continues conversation.
- Keep spacing, styling, and readability consistent.

## Implementation Notes
- Clear prior assistant suggestions when the user sends a new message.
- Keep suggestion chips compact and unobtrusive.

## Completion Criteria
- Suggestion UI remains clean and non-repetitive.
- Suggestions do not clutter older message history during active conversation.
